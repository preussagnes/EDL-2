///Programa em c++

#include <iostream>
using namespace std;


int main()
	{
	int N;
	
	cin >> N;
	int Array[N];
	
	
	//Impossivel fazer Array.Length, Logo a variavel N deve ser guardad a fim q se percorar o array.
	cout << "O array tem tamnho" << N <<"\n";
	
	//Ou se dividir o tamanho todo do array pelo tamanho do seu tipo.
	cout << "O array tem tamnho" << sizeof(Array)/sizeof(int) <<"\n";
	
	
	return(0);
	}
	

	
///Programa em C#

using System;

public class Program
{
	public static void Main(String [] agrs)
	{
		Console.WriteLine("Escreva o tamanho do array:");
		int N = Convert.ToInt32(Console.ReadLine());
		int[] Array = new int[N];
		
		Console.WriteLine("O tamanho do arry eh:");
		Console.WriteLine(Array.Length);
		
	}
}



Se nota duas maneiras diferente de se trabalhar com arrays, em c++ um array é apenas um alocamento de memória continuo,
mas algo problematico em trabalhar com arrays em c++ é conseguir o tamanho do arry em casos quando são alocados por variáveis,
sabendo-se apenas o tamanho deste arry em tempo de execução, em c#, á uma forma de se obter o "tamanho" do arry já declarado, pois em
c#, alem de ser um espaço continuo na memória, arrays também são objetos/classes, por tanto tendo seus respectivos campos. Logo é possível saber
o tamanho de um array simplesmente acessando o campo Length.
Analisando a situação, percebe-se uma problematica na linguagem c++, "Como descobrir o tamanho de um array generico c++ ?"
A primeria resposta a se pensar é sempre guardando a variavel (no exemplo N) e a partir dessa informação será possivel sempre saber o tamanho do arry,
com tudo, isso pode conter falta de expresão ao progamador inexperiente, pois se por acaso ele usar alguma função que tenha como paramentro esse arry, ele precisamente 
necessitará passar essa variavel que guarda o tamanho do arry como paramento também. Há uma forma de contornar este "incomodo"
basta usar sizeof no indentificador do array que retornarar o seu comprimento multiplicado pelo tamanho do tipo do array, e en seguida basta dividir pelo tamanho
do tipo assim obtendo seu tamanho, sendo uma forma possivel porrem nada intuitiva
